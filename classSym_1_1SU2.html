Detailed Description

template<typename Kind, typename Scalar = double>
class Sym::SU2< Kind, Scalar >

Class for handling a SU(2) symmetry of a Hamiltonian without explicitly store the Clebsch-Gordon coefficients but with computing -symbols.

\describe_Scalar

Note
An implementation for the basic symbols is used from SU2Wrappers.h. Currently, only the gsl-implementation can be used, but any library which calculates the symbols can be included. Just add a wrapper in SU2Wrappers.h.
